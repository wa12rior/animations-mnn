Plik wrzucamy do naszego ulubionego programu do edycji grafiki wektorowej.

Jeśli ktoś nie ma takiego programu, polecam Figmę :)

Następnie dzielimy odpowiednie części na 4 podgrupy: chłopak, szkielet roweru, koło lewe oraz prawe

Skrót klawiszowy Ctrl + G

Zaznaczamy cały element frame po lewej i w panelu po prawej stronie eksportujemy na dole na pasku klikamy export, jako svg, klikamy 3 kropki i wszystkie opcje mają być zaznaczone.

Tworzymy następnie plik index.html (rozszerzenie html istotne!) w ulubionym edytorze tekstowym

i wrzucamy do środka nasz plik svg

Tworzymy plik style.css w tym samym katalogu co html i wklejamy poniższą zawartość


        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }

        html, body, .container {
            width: 100%;
            height: 100%;
        }

        .container {
            display: flex;
            justify-content: center;
            align-items: center;
        }
